ㅤㅤO Sistema Numérico Binário consiste de 2 números, o "0" e o "1", e a junção deles faz com que seja possível representar diversos números, assim como representado na tabela acima. Por exemplo quando o 1 está na última posição, o número vai ser ímpar, e quando 0 estiver na última posição o número vai ser par. O Sistema Binário é a linguagem utilizada nos computadores, a linguagem de sim e não, 1 sendo sim e 0 sendo não, por mais simples que pareça é possível fazer diversas coisas complicadas por meio deste simples processo.

Regra da Decomposição

Para decompor primeiro é necessário saber como são representados os números dos Sistemas:

Tabela de Conversão

Conversão ㅤㅤ|ㅤㅤ Procedimento
Base 10 → Base X | Divisões sucessivas pelo número da Base X e tomar os restos.
Base X → Base 10 | Decompor o número em potências de X conforme posição de seus dígitos.
Base 2 → Base 8 e Base 16 | Separar o número binário em grupos de três(Base 8) ou 4(Base 16) e substituir os grupos pelos respectivos dígitos octais ou hexadecimais, conforme as tabelas de equivalência dadas
Base 8 e 16 → Base 2 | Cada dígito octal ou hexadecimal deve ser substituído pelso grupos de 3 ou 4 dígitos binários, cconforme as tabelas de equivalência dadas
Base X → Base Y | Base X → Base 10 → Base Y

Para fazer a conversão, são feitas divisões sucessivas, e o jeito como é feito é por meio da regra da "Escadinha".
